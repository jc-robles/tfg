Welcome to IMU Data Process

  • Click on Upload test file button to add a dataset to be processed
  • Click on Create test type button to add a new test type
  • Click on Delete test type button to delete an existing test type

Select a CSV file

A csv file containing the following structure must be selected:
  • The header should have this order: timestamp, accelerometer x, accelerometer y, accelerometer z, gyroscope x , gyroscope y, gyroscope z, magnetometer x, magnetometer y, magnetometer z.
  • The row following the header must be indicated with the units of each column.
  • If the units are not available, each cell can be left empty or with a N/A value.
File is required

Create test type

  • The name to which the new type of test refers must be entered.
    Name is required
  • All the output fields of the new test type must be indicated. Output fields can be :
    • Alphanumeric: when the output is to be displayed in a text.
    • Data Array: when the output data is to be displayed in a graph.
    Name field is required
    Field is required
    Data type is required

  • Each grouping corresponds to a graph. All the fields that are indicated with the same grouping will be shown in the same output graph.

    With this option you can create or delete the different types of groupings.

  • File is required
    You must select the python file that contains the code that will be executed when the data is processed.
